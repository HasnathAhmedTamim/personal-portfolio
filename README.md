# PORTFOLIO - Hasnath Ahmed Tamim

A personal portfolio built with React + Vite and Tailwind CSS. This template provides a minimal setup to develop a fast, modern frontend with HMR and sensible defaults.

## Features

- Vite + React for fast development with HMR
- Tailwind CSS for utility-first styling
- Simple component structure in `src/components`
- Project data lives in `public/projects.json`

## Getting started

Prerequisites: Node.js (16+ recommended) and npm or yarn.

Install dependencies:

`npm install`

Run the dev server with HMR:

`npm run dev`

Build for production:

`npm run build`

Preview the production build locally:

`npm run preview`

## Project structure (key files)

- `index.html` — app entry HTML
- `package.json` — scripts and dependencies
- `vite.config.js` — Vite configuration
- `tailwind.config.js` and `postcss.config.js` — Tailwind setup
- `public/projects.json` — project data used by the portfolio
- `src/` — React app source
	- `src/main.jsx` — app bootstrap
	- `src/App.jsx` — top-level component
	- `src/components/` — UI components (Navbar, Home, About, PortfolioProjects, Reviews, Contact, Footer, etc.)

## Notes

- This repo is intentionally minimal — customize components and styling to match your brand.
- If you use TypeScript or additional tooling, update `package.json` and config files accordingly.
